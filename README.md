## Featured Projects

### Smart Gardening Assistant  
End-to-end IoT system integrating sensors with an ESP32 microcontroller and a CNN-based disease detection model.  
- Achieved 92% classification accuracy across 38+ plant diseases  
- Reduced water usage by ~35% through automation  
- Combines hardware and ML for real-world deployment  

---

### Heart Failure Prediction  
Deep learning model for clinical risk prediction.  
- Built a 5-layer neural network  
- Achieved 87% accuracy with 0.91 ROC-AUC  
- Focus on interpretable and reliable predictions  

---

### Medical Image Analysis  
Multi-class classification using transfer learning.  
- Implemented ResNet-18 architecture  
- Achieved 89% accuracy on 10,000+ medical images  
- Optimized for performance and generalization
